#TextInput notifications
iOS9 introduced the ability to quickly reply to a notification (either local or remote) with a text input, like the Messages app does. This sample shows how to implement it.

#Setup
This project requires Xcode 7. Just open the provided .xcodeproj.  
The app schedules a notification one minute ahead of the current time.
